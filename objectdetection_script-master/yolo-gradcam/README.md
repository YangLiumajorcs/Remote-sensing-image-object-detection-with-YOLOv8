## 注意事项

2. yolov7是在2023.10.1号的版本进行编写和测试的。
3. yolov8是在2024.1.31号的版本进行编写和测试的。
4. yolov9是在2024.3.7号的版本进行编写和测试的。
5. 建议在新版本下进行使用，旧版本可能会有报错，需要自行解决。
